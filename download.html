Install Guide

Android 설치 가이드

APK 파일 설치가 처음이신가요? 아래 단계를 따라하세요.

1

APK 다운로드 — 위 버튼을 눌러 APK 파일을 다운로드합니다.

2

출처 허용 — 설정 → 보안 → "알 수 없는 출처의 앱 설치"를 허용합니다. (최초 1회)

3

설치 실행 — 다운로드된 APK 파일을 탭하여 설치를 진행합니다.

4

앱 실행 — 설치 완료 후 앱을 열고 회원가입을 진행하세요. 바로 게임을 시작할 수 있습니다!

FAQ

자주 묻는 질문

FunS 앱은 무료인가요? +

네, FunS 앱은 완전히 무료로 다운로드하고 사용할 수 있습니다. 앱 내 구매나 숨겨진 비용은 없으며, 게임을 즐기고 토큰을 획득할 수 있습니다.

어떤 기기에서 사용 가능한가요? +

FunS 앱은 Android 8.0 이상을 지원합니다. APK를 직접 설치하거나 웹 브라우저에서 접속하여 PC와 모바일 모두에서 이용할 수 있습니다.

게임에서 획득한 토큰은 어떻게 사용하나요? +

게임에서 획득한 FUNS 토큰은 앱 내 FunSwap DEX에서 다른 암호화폐로 스왑하거나, 지갑에 보관했다가 필요할 때 사용할 수 있습니다.

보안은 안전한가요? +

FunS 앱은 블록체인 기술을 기반으로 하며, 사용자의 지갑과 자산을 안전하게 보호합니다. 개인 키는 사용자의 기기에만 저장되며, 우리는 접근할 수 없습니다.
