Features

왜 FUNS Talk인가요?

이메일도, 전화번호도 필요 없습니다. 내 대화는 나만 볼 수 있고, 채팅창에서 바로 송금까지.

🔐

완벽한 E2E 암호화(군사급 암호화)

보내는 사람과 받는 사람만 메시지를 읽을 수 있습니다. 서버조차 내용을 알 수 없는 군사급 암호화.

👛

지갑으로 간편 가입

앱 설치 시 지갑이 자동 생성됩니다. PIN이나 지문으로 로그인하세요. MetaMask 등 기존 지갑 연결도 OK.

💸

채팅창에서 바로 송금

대화 중에 BNB, FunS 토큰을 바로 보낼 수 있습니다. /send 한 줄이면 끝. 복잡한 주소 복붙은 필요 없어요.

🎫

토큰 게이티드 채널

특정 NFT나 FunS 토큰 보유자만 입장 가능한 프리미엄 채널. 나만의 커뮤니티를 만들어보세요.

⛽

가스비 걱정 제로

기본 채팅은 완전 무료입니다. 블록체인 수수료? FUNS Talk이 대신 내드려요. 카톡처럼 그냥 쓰세요.

👥

그룹 채팅 & 파일 공유

암호화된 그룹 채팅에서 파일과 미디어를 안전하게 공유하세요. 슈퍼챗으로 메시지를 하이라이트할 수도 있어요.

How to Use

시작은 정말 간단해요

카카오톡만큼 쉽습니다. 블록체인은 뒤에서 알아서 돌아갑니다.

1

앱 설치

다운로드하면 지갑이 자동으로 만들어져요. 별도 설정 없이 바로.

2

PIN / 생체인증

지문이나 PIN 하나로 로그인. 시드 구문 외울 필요 없어요.

3

친구 추가 & 채팅

지갑 주소나 닉네임으로 친구 추가. 모든 대화는 자동 암호화됩니다.

4

송금 & 커뮤니티

채팅창에서 바로 송금하고, 토큰 게이티드 채널에 참여하세요.

FunS Token

FunS 토큰으로 할 수 있는 것

FUNS Talk 안에서 다양하게 쓰이고, 쓸 때마다 일부가 소각되어 점점 희소해집니다.

🪙 이런 곳에 쓸 수 있어요

프리미엄 메시지 (읽음확인, 예약전송) 50% 소각
슈퍼챗 (그룹 하이라이트 메시지) 30% 소각
토큰 게이티드 채널 입장 70% 소각
프로필 NFT 뱃지 구매 100% 소각
인앱 스티커 & 이모지팩 50% 소각

💬 채팅창에서 바로 실행

/send 친구에게 FunS 송금
/tip 메시지에 팁 붙이기
/burn 자발적 소각 (커뮤니티 기여)
/gate 채널을 토큰 게이트로 전환
/balance 내 FunS 잔액 확인

🔥 사용할수록 가치가 올라가는 구조

사용자 활동 증가 → FunS 소비 증가 → 소각량 증가 → 공급 감소 → 토큰 가치 상승 🔁
